# Method-Class-Assignment

Description

This is a simple C# console application that demonstrates:

How to create a class with a void method.

How to pass parameters to the method, including using named parameters.

How to perform a simple math operation inside the method.

How to display results to the console.

Features

Class Creation: MathClass contains a method called DoMath.

Method Functionality:

Takes two integers as input.

Performs a math operation on the first integer (multiplication by 2).

Displays the second integer.

Named Parameters: The method can be called using parameter names for clarity.

Console Output: Shows results of the math operation and the second number.

Usage

Run the program.

The console will display the results of the math operation on the first number and the value of the second number.

Example outputs:

Result of math operation on first number: 10
Second number is: 10
Result of math operation on first number: 14
Second number is: 20
Press any key to exit...
